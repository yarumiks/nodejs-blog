Guide to Javascript Regular Expressions(RegEx)

Regular expressions provide a powerful way to perform pattern matching on certain characters within strings of text. They offer a concise syntax to carry out complex tasks that otherwise would require lengthy code. Syntax : /pattern/flags Here the flags are optional- I have discussed this in last. Meta-characters Meta-characters are characters with a special meaning. There are many meta-characters but these you must know. \d — Match any digit character ( same as [0-9] ). \w — Match any word character. A word character is any letter, digit, and underscore. (Same as [a-zA-Z0–9_] ) i.e alphanumeric character. \s — Match a whitespace character (spaces, tabs, etc). \t — Match a tab character only. \b — Find a match at the beginning or end of a word. Also known as a word boundary. . — (period) Matches any character except for the newline. \D — Match any non-digit character (same as [^0–9]). \W — Match any non-word character (Same as [^a-zA-Z0–9_] ). \S — Match a non-whitespace character. First example with regex The pattern portion above starts with an ^ indicating the beginning of a string. The \d indicates a digit followed by {2} meaning 2 consecutive digits. The $ indicates the end of a string. So, this pattern will attempt to find exactly 2 consecutive digits from the beginning to the end of a string. var regex = /^\d{2}$/; var str1 = “30”; console.log( regex.test( str1 ) ); //Output: true var str2 = “4s5”; console.log( regex.test( str2 ) ); //Output: false search() When searching in a given string, you can specify the pattern you’re looking for. Below we search for the first instance of a lower-case character [a-z] followed by one or more digits [0–9]+. var str = “Shahin Mahmud012”; var regex = /[a-z][0–9]+/; console.log(`The postion of d(digit) = ${str.search(regex)}`); //Output: The postion of d(digit) = 12 replace() The next expression parses the string, removes all the digits [\d] from it, and returns the remaining string. Note that the g is a global modifier which means find all matches and don’t stop after the first match is found.
